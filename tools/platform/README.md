# 桌宠精灵表生成器

一个基于 Web 的工具，用于创建和编辑桌宠精灵表。

## 功能特性

- **三种上传模式**：
  - 单张上传：逐帧上传图片，精细控制每个动画
  - 整张上传：直接上传已制作好的精灵表
  - 智能生成：上传一张图片，自动生成多帧动画

- **支持左右行走分离**：行1为向右行走，行2为向左行走
- **实时预览**：上传后立即查看精灵表效果
- **导出下载**：生成标准格式的 PNG 精灵表
- **响应式设计**：支持桌面和移动设备

## 使用方法

### 方式一：直接打开
双击 `index.html` 文件，在浏览器中打开生成器。

### 方式二：使用启动脚本
运行 `launch_generator.ps1` PowerShell 脚本。

## 精灵表格式

生成的精灵表格式：
- 4 列 × 8 行
- 每行对应一个状态：Idle、Walk Right、Walk Left、Sleep、Eat、Happy、Startled、Sit
- 每行包含 4 帧动画
- 品红色 (255, 0, 255) 为透明色

### 布局说明

| 行号 | 状态 | 说明 |
|------|------|------|
| 0 | Idle | 待机 |
| 1 | Walk Right | 向右行走 |
| 2 | Walk Left | 向左行走 |
| 3 | Sleep | 睡觉 |
| 4 | Eat | 进食 |
| 5 | Happy | 开心 |
| 6 | Startled | 受惊 |
| 7 | Sit | 坐下 |

## 输出设置

- **帧宽度**：每帧的宽度（默认 32px）
- **帧高度**：每帧的高度（默认 32px）
- **每行帧数**：每行的帧数（默认 4）

## 使用步骤

1. 选择上传模式
2. 上传图片素材
3. 调整输出设置（可选）
4. 点击"生成精灵表"
5. 点击"下载 PNG"保存
6. 将生成的 `pet.png` 替换项目中的 `assets/pet.png`
7. 修改 `config/config.json` 中的 `rows` 为 8

## 技术栈

- HTML5 Canvas
- 纯 JavaScript，无依赖
- CSS3 动画和渐变

## 注意事项

- 建议使用透明背景的 PNG 图片
- 每帧尺寸应保持一致
- 推荐尺寸：32x32 或 64x64 像素
- 向左行走可以是向右行走的镜像，也可以是独立绘制
