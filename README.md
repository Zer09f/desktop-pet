# 🐾 桌面宠物 (Desktop Pet)

运行在 Windows 桌面上的像素风小宠物，以透明窗口形式存在，能在屏幕底部自由漫步并与用户互动。

## 快速开始

```bash
# 编译运行
cargo run --release

# 停止：关闭窗口 或 Ctrl+Shift+Q
```

## 交互说明

### 鼠标操作

| 操作 | 状态 | 效果 |
|------|------|------|
| 鼠标悬浮到宠物上方 | 🧘 待机 (Idle) | 宠物停止走动，站立不动 |
| 鼠标从宠物上移开 | 🏃 奔跑 (Walk) | 宠物恢复自动左右行走 |
| 左键点击宠物 | 😱 受惊 (Scared) | 惊吓动画，2秒后自动恢复 |
| 右键点击宠物 | 🪑 坐下 (Sit) + 菜单 | 坐下动画，3秒后自动恢复 |

### 自动行为（屏幕位置触发）

| 位置 | 状态 | 效果 |
|------|------|------|
| 屏幕最左侧边缘 | 😴 睡觉 (Sleep) | 宠物停在左边睡觉，鼠标悬浮可唤醒 |
| 屏幕最右侧边缘 | 😊 开心 (Happy) | 宠物停在右边开心，鼠标悬浮可唤醒 |
| 屏幕中间区域 | 🚶 行走 (Walk) | 宠物自动左右来回走动 |

### 右键菜单

| 菜单项 | 功能 |
|--------|------|
| 🎨 换皮肤 | 选择本地图片替换宠物精灵表 |
| ✨ 去除白底 | 自动将图片白色/浅色背景变透明（备份原图为 .bak） |
| 🍖 喂食 | 增加好感度（可配置增量） |
| 🥰 摸摸 | 增加好感度 |
| 🎮 玩耍 | 互动玩耍 |
| ⚙ 设置 | 打开设置（预留） |
| ❌ 退出 | 关闭桌宠 |

### 快捷键

| 快捷键 | 功能 |
|--------|------|
| Ctrl+Shift+Q | 立即退出桌宠 |

## 精灵表规格

文件位置：`assets/pet.png`

- 尺寸：128×256 像素
- 布局：4 列 × 8 行
- 每帧：32×32 像素

| 行号 | 状态名 | 触发条件 | 说明 |
|------|--------|----------|------|
| Row 0 | Idle（待机） | 鼠标悬浮 | 默认站立不动 |
| Row 1 | WalkRight（向右跑） | 自动行走方向朝右 | 向右移动动画 |
| Row 2 | WalkLeft（向左跑） | 自动行走方向朝左 | 向左移动动画 |
| Row 3 | Sleep（睡觉） | 到达屏幕左边缘 | 停止移动，睡觉动画 |
| Row 4 | Happy（开心） | 到达屏幕右边缘 | 停止移动，开心动画 |
| Row 5 | Scared（受惊） | 左键点击 | 持续2秒后恢复 |
| Row 6 | Sit（坐下） | 右键点击 | 持续3秒后恢复 |
| Row 7 | （备用） | - | 暂未使用 |

## 配置文件

路径：`config/config.json`

```json
{
  "pet_width": 128,
  "pet_height": 128,
  "walk_speed": 25.0,
  "image_path": "assets/pet.png",
  "frames_per_row": 4,
  "rows": 8,
  "anim_fps": 6.0,
  "feed_affection_gain": 5,
  "affection_decay_per_hour": 1
}
```

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| pet_width | u32 | 128 | 宠物窗口宽度（像素） |
| pet_height | u32 | 128 | 宠物窗口高度（像素） |
| walk_speed | f32 | 25.0 | 行走速度（像素/秒） |
| image_path | String | assets/pet.png | 精灵表文件路径 |
| frames_per_row | u32 | 4 | 精灵表每行帧数 |
| rows | u32 | 8 | 精灵表总行数 |
| anim_fps | f32 | 6.0 | 动画播放帧率 |
| feed_affection_gain | u32 | 5 | 每次喂食增加的好感度 |
| affection_decay_per_hour | u32 | 1 | 每小时好感度自动衰减值 |

## 去除白底功能

适用场景：精灵图片有白色/浅色背景需要透明化。

使用方法：
1. 右键点击宠物 → 选择「✨ 去除白底」
2. 程序自动检测白色及接近白色的像素并设为透明
3. 处理前自动备份原图为 `pet.png.bak`
4. 边缘半透明像素做渐变过渡，避免锯齿
5. 处理完成后自动重新加载精灵

技术细节：
- 阈值：RGB 最低通道 > 230 的像素完全透明
- 渐变区间：RGB 最低通道 190~230 之间做 alpha 渐变

## 状态机流程

```
                    鼠标悬浮
    ┌──────────────┐ ──────────> ┌──────────┐
    │  WalkRight   │             │   Idle   │
    │  (向右跑)    │ <────────── │  (待机)   │
    └──────────────┘  鼠标离开   └──────────┘
           │                         │
     到达右边缘                 到达左边缘(方向反转后)
           │                         │
           ▼                         ▼
    ┌──────────────┐           ┌──────────────┐
    │    Happy     │           │    Sleep     │
    │   (开心)     │           │   (睡觉)     │
    └──────────────┘           └──────────────┘
           │                         │
           └── 鼠标悬浮可唤醒 ────────┘

    左键点击 → Scared (受惊, 2秒) → 自动恢复
    右键点击 → Sit (坐下, 3秒) → 自动恢复
```

## 技术实现

- **语言**：Rust
- **窗口**：Win32 API (WS_EX_LAYERED + LWA_COLORKEY 颜色键透明)
- **渲染**：32位 DIB Section + BitBlt
- **透明色**：洋红色 (0xFF00FF) — 精灵中不会出现的颜色
- **精灵绘制**：仅绘制 alpha > 0 的像素，透明部分保留洋红色背景
- **窗口层级**：WS_EX_TOPMOST 始终置顶
- **DPI感知**：SetProcessDPIAware，适配高分辨率屏幕
